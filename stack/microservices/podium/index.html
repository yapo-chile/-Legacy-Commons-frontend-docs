# Podium

There are several approaches to working with microfrontend (opens new window), but after studying all the benefits we have decided to work with Podium (opens new window).

you can find more information in our confluence documents (opens new window)

Podium is a library that allows us to build a site by dividing it into segments, for example we can create a header, a content section and a footer, all destined to work together, but developed and executed in isolation.

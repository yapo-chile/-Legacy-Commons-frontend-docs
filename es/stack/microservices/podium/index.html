# Podium

Existen varias maneras para trabajar con microfrontend (opens new window), pero después de estudiar los beneficios decidimos trabjar con Podium (opens new window).

tu puedes encontrar más información en nuestros documentos de confluence (opens new window)

Podium es una librería que nos permite construir un sitio dividido en segmentos. Por ejemplo, podemos crear una cabecera, una sección de contenido y un pie de página, todos destinados para trabajar juntos, pero desarrollados y ejecutados de manera aislada.
